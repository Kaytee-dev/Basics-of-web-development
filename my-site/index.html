Git commmands

Below are some of the git commands that I know

  1. Git clone :- used to duplicate a repository on a local machine
  2. Git status :- used to check the status of commits in a repository or branch
  3. Git add :- used to queue changes that needs to be to commited to the remote server
  4. Git checkout :-to change from one branch to another
    5. syntax git checkout "gh-pages" 
    the above syntax command will change the branch to gh-pages
  5. Git push :- to forward changes made on the local to remote repository
  6. Git pull :- to download all the commits made to a repository

Browser sync syntax

use the syntax below on cmd to access a local host where changes made to your html files will be updated immediately
browser-sync start --server --directory --files "*"
